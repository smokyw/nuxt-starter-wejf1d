## Локальная разработка

1. Создать `.env` файл по примеру из `.env.example`
1. Установить зависимости

   ```sh
   pnpm install
   ```

1. Запустить `dev`-сервер

   ```sh
   pnpm dev
   ```

## Продакшн

Сборка в продакшн

```sh
pnpm build
```

Превью продакшн билда

```sh
pnpm preview
```

## Тестирование

Проверка типов

```sh
pnpm typecheck
```

## Линт

Запуск ESLint

```sh
pnpm lint
```

Запуск Prettier

```sh
pnpm prettier
```
